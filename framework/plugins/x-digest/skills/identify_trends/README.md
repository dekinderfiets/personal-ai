# Identify Trends

Analyze content to find topics.

## Purpose

Clusters discussions into trends and themes.

## Inputs

| Input | Required | Description |
|-------|----------|-------------|
| `filtered_content` | Yes | High-quality tweets |

## Instructions

### Step 1: Topic Analysis

Group discussions by shared themes. Identify emerging patterns.

### Step 2: Categorize

- **Trending**: High attention.
- **Notable**: Important but steady.
- **Insights**: Key takeaways.
